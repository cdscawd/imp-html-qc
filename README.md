## 开发
  IDE: [WebStorm 2017.1.3](https://www.jetbrains.com/webstorm/)
  
  NodeJS: [v6.10.0](https://nodejs.org)

### 安装依赖

在当前目录下运行以下命令:

    $ npm install --global gulp
    npm install

### 开始开发

#### 在当前目录下运行以下命令:
   开发环境打包

    $ gulp

   运行本地环境实例
   
    $ gulp serve
  
   备用运行端口
      
    $ gulp serve:dist
